# Search Algorithms

## Clonar el repositorio

```
git clone git@github.com:MiguelCuevasss/search_algorithms.git
cd search_algorithms
```

## Instalar dependencias

```
pip install pytest pytest-benchmark
```

## Ejecutar unit testing

Para ejecutar todos los unit tests:

```
pytest
```

Para ejecutar solo el unit testing de búsqueda lineal:

```
pytest test_linear_search.py
```

Para ejecutar solo el unit testing de búsqueda binaria:

```
pytest test_binary_search.py
```


## ¿Cómo ejectuar el Benchmark?

    Para ejecutar el Benchmark ingresa el siguiente codigo desde la carpeta en la que se clonó el Repositorio:

                                "python -m pytest benchmarking.py"
